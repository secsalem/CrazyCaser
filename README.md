# CrazyCaser
CrazyCaser: A case variation tool that transforms text into a mix of upper and lower cases

Installation:

```
git clone https://github.com/secsalem/CrazyCaser.git
```
```
chmod +x CrazyCaser.py
./CrazyCaser.py
```

Usage:

```
python CrazyCaser.py
OR
./CrazyCaser.py
Enter a word: data
```
